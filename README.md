# Sunstrider Agency

Portail institutionnel fictif du Royaume de Quel'Thalas, construit avec Vite, React et Tailwind CSS.

## Site publié

https://melui3.github.io/sunstrider-agency/

## Lancer le projet

```bash
npm install
npm run dev
```

Le site tourne ensuite sur `http://127.0.0.1:5173/`.

## Build

```bash
npm run build
```

## Stack

- Vite
- React
- Tailwind CSS
- lucide-react
